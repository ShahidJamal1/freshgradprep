# Project Setup Guide 🚀

## Prerequisites

Make sure you have installed:

- Node.js
- npm
- Git
- IDE of your choice (VSCode, Cursor, Claude Code, etc.)

## Which branch to use
- never make any changes to the `main` branch
- always clone the `dev` branch, and then create your own branch to make changes.

## Fork the repository
- Please fork the repository first,
- clone your fork locally,
- install dependencies using `npm install`,
- run the project using `npm run dev`.
- Once the app is running, pick an assigned issue, create a separate branch for your work, and raise a PR once your changes are ready.
